# agentic-eng

Laboratório pessoal de estudos sobre Engenharia de Agentes, LLMs, workflows com IA, engenharia de prompts, automações inteligentes e sistemas orientados por linguagem natural.

Este repositório documenta minha jornada de aprendizado sobre como Large Language Models podem ser usados além de interfaces simples de chat: como assistentes de tarefas, sistemas de raciocínio, orquestradores de workflows e componentes de produtos inteligentes.

## Objetivo

O objetivo deste repositório é construir conhecimento prático sobre:

- Fundamentos de LLMs
- Engenharia de prompts
- Uso de ferramentas por modelos
- Workflows com agentes
- RAG, Retrieval-Augmented Generation
- Avaliação e confiabilidade
- Automações com IA
- Pensamento de produto aplicado a sistemas inteligentes

## Foco atual

O foco atual é entender os fundamentos do comportamento dos LLMs e praticar padrões de prompt por meio de pequenos experimentos documentados.
